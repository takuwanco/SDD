## ② 企画・要件定義工程

### 目的

2つの独立したドキュメント不整合を同一 PR で解消する。

1. **Issue #65**: `spec-ai-writer/QUICKSTART.md` は `README.md` と内容が重複しており、かつ README から有効な導線がない。二重管理による乖離リスクを解消するため廃止する。
2. **Issue #66**: `pyproject.toml` / `frontend/package.json` はバージョン `1.0.3` を示しているが、`CHANGELOG.md` / `CHANGELOG_ja.md` では同バージョンが `Unreleased` のまま残っており、バージョン比較表にも行が存在しない。

### 背景

#### Issue #65 — QUICKSTART.md の廃止

`QUICKSTART.md` に含まれる情報を `README.md` と比較した結果、以下が確認できた。

| 観点 | 状況 |
|------|------|
| QUICKSTART 固有の有用情報 | OpenRouter / Ollama の `.env` 例が簡潔にまとまっているが、README.md の「LLMプロバイダーの設定」セクションでも同等の情報を提供済み |
| README に存在し QUICKSTART に欠ける情報 | `uv sync --extra dev`、Web UI 起動手順、`DATA_DIR` / `APP_ENV` など網羅的な `.env` 説明 |
| README からの導線 | ディレクトリ構成に記載なし、目次やリンクも存在しない |

#### Issue #66 — バージョン表記の不整合

Issue #60 で全ファイルを `1.0.3` に統一した際、CHANGELOG のリリース日記入とバージョン比較表への追加が漏れた。コードは `1.0.3` として動作しているため、CHANGELOG を実態に合わせてリリース済み扱いに修正する。

### 修正スコープ

| 優先度 | 問題 | 対象ファイル |
|-------|------|------------|
| 高 | QUICKSTART.md が README と重複・導線なし | `spec-ai-writer/QUICKSTART.md`（削除） |
| 中 | CHANGELOG の 1.0.3 が Unreleased のまま | `CHANGELOG.md` / `CHANGELOG_ja.md` |
| 中 | バージョン比較表に 1.0.3 の行がない | `CHANGELOG.md` / `CHANGELOG_ja.md` |
