## 動画IDの更新について

現在、一部の動画IDが `VIDEO_ID_X` の形式になっています。実際の動画IDを取得するには、以下の方法があります：

1. **再生リストから取得**: [再生リストページ](https://www.youtube.com/watch?v=0U9PPY1tGU8&list=PLv-ekZfLyxXF0bQKFJfdQaYUquBOVXPPa) を開き、各動画のURLから動画IDを取得
2. **YouTube Data APIを使用**: プログラムで再生リストから動画情報を取得
3. **手動で更新**: 各動画のURLから動画IDを取得し、`VIDEO_ID_X` を実際の動画IDに置き換え

**動画IDの取得方法**:
- YouTubeの動画URLから取得：`https://www.youtube.com/watch?v=VIDEO_ID` の `VIDEO_ID` 部分
- 例：`https://www.youtube.com/watch?v=0U9PPY1tGU8` の場合、動画IDは `0U9PPY1tGU8`
